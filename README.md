# Contact CRUD App

Frontend coding test project built with React, TypeScript, Vite, Redux Toolkit, and Vitest.

## Features

- Load 30 contacts from Random User API.
- Add, edit, delete, search, and refresh contacts.
- Redux Toolkit for state management.
- Responsive UI for desktop and mobile.
- Unit tests for Redux logic and API data mapping.

## Tech Stack

- React
- TypeScript
- Vite
- Redux Toolkit
- React Redux
- Vitest
- Testing Library

## Run Locally

```bash
npm install
npm run dev
```

## Test

```bash
npm run test:run
```

## Build

```bash
npm run build
```
